# Destinations
A Teleportation plugin for Sponge. Checkout the [Wiki](https://github.com/mmonkey/Destinations/wiki) details.

### Commands
**Homes**
* `/home [name]` alias: `/h`
* `/sethome [-f] [name]`
* `/delhome <name>`
* `/listhomes [page]` alias: `/homes`

**Warps**
* `/warp [name]` alias: `/w`
* `/setwarp <name>`
* `/delwarp <name>`
* `/listwarps [page]` alias: `/warps`
